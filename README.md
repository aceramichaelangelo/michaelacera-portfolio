## Planned Tech Stack

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Language:** TypeScript
- **Deployment:** Vercel (recommended)

## Quick Start (After Implementation)

Once development begins, the typical workflow will be:

1. Install dependencies
2. Run the development server
3. Build for production
4. Deploy to Vercel

Detailed commands will be added to this README during Phase 1 of the roadmap.

## Design Principles

- Modern dark UI with glassmorphism accents
- Bento grid layouts for projects and skills
- Mobile-first, fully responsive
- High accessibility (WCAG 2.1 AA target)
- Fast loading and optimized assets
- Smooth, purposeful animations
