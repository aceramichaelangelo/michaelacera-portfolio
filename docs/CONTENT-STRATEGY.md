## Section 6: Featured Project — Native Flora Detection and Management System

### Tagline
Environmental monitoring powered by computer vision and geoanalytics.

### Problem Statement (recommended copy)

> Identifying and managing native flora species in the field is challenging without specialized botanical knowledge. Manual identification is slow, error-prone, and makes it difficult to map species distribution across geographic areas.

### Solution (recommended copy)

> A cross-platform system combining Flutter mobile apps and a web dashboard that uses YOLO object detection models to identify native flora species from images, classify them automatically, and map their locations using geoanalytics — enabling researchers and environmental workers to monitor biodiversity more efficiently.

### Features
- Real-time native flora species identification via camera or image upload
- YOLO-based object detection and classification models
- Interactive species mapping with geoanalytics integration
- Cloud database for species records and detection history
- Cross-platform access via Flutter mobile and web interfaces
- AI-assisted development workflow for rapid iteration

### Technologies
Flutter · Cloud Database · Computer Vision · YOLO Object Detection · Geoanalytics · AI-Assisted Development

### Development Process
1. Research native flora species and environmental monitoring needs
2. Plan system architecture for mobile, web, and cloud components
3. Design UI/UX in Figma for field worker and admin interfaces
4. Train and integrate YOLO detection models for flora classification
5. Develop Flutter app with camera integration and cloud sync
6. Build web dashboard for species management and mapping
7. Test detection accuracy and usability in field conditions
8. Deploy cloud database and application services

### Impact
- Streamlines native flora identification for non-specialists
- Enables data-driven environmental monitoring and biodiversity tracking
- Demonstrates integration of mobile development, AI, and geoanalytics
- Represents capstone-level work combining multiple technology domains
